# Atom

GitHubが提供する高機能テキストエディタ、Atomを紹介します。

## Atom とは

Atomとは冒頭の一言にあるように、GitHubが開発しているオープンソースのテキストエディタです。

拡張機能やテーマなどによるカスタマイズ性や、安定した動作が魅力的です。
